Case Study: i586 Mandriva 2006, R14

The following RPMs were installed via URPMI from the contrib media source (consult Easy URPMI for information on how to set up your URPMI media sources). Then copy the mexopts.sh from your ${MATLAB}/bin directory into your working directory.

gdal-1.2.6-3mdk
libgdal1-1.2.6-3mdk
libgdal1-devel-1.2.6-3mdk

Then the bottom of mexopts.sh was modified to read 

	CLIBS="$CLIBS -lgdal"

Save these edits and invoke make from the unix command line. Because the RPMs install in the normal locations, i.e. /usr/lib and /usr/include, the edits to mexopts.sh are pretty minimal.
